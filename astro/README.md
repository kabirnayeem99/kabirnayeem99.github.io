# Astro Workspace

Astro is the only site pipeline in this repository.

## Commands

```bash
npm ci
npm run dev
npm run lint
npm run check
npm run test
npm run build
```

## Notes

- Content files live in `src/data/site-content/` (split by shared sections and per-page JSON).
- `robots.txt` and `sitemap.xml` are generated by Astro routes.
- Deployment uploads `dist/` to GitHub Pages.
