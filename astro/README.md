# Astro Workspace

Astro is the only site pipeline in this repository.

## AI tooling note

`lean-ctx` and `jcodemunch` are configured at repository root. Run MCP/indexing commands from the root directory (not `astro/`), using:

```bash
CODE_INDEX_PATH="$PWD/.code-index" jcodemunch-mcp index "$PWD"
```

## Commands

```bash
npm ci
npm run dev
npm run lint
npm run check
npm run test
npm run build
```

## Notes

- Content files live in `src/data/site-content/` (split by shared sections and per-page JSON).
- `robots.txt` and `sitemap.xml` are generated by Astro routes.
- Deployment uploads `dist/` to GitHub Pages.
