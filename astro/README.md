# Astro Workspace

Astro is the only site pipeline in this repository.

## Commands

```bash
npm ci
npm run dev
npm run lint
npm run check
npm run test
npm run build
```

## Notes

- `site-content.json` lives in `src/data/site-content.json`.
- `robots.txt` and `sitemap.xml` are generated by Astro routes.
- Deployment uploads `dist/` to GitHub Pages.
