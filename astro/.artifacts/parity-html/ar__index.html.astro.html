أَعْمَلُ كَمُهَنْدِسِ بَرْمَجِيَّاتٍ، وَأُرَكِّزُ غَالِبًا عَلَىٰ تَطْوِيرِ تَطْبِيقَاتِ ٱلْمُوبَايِل. وَلَدَيَّ بَعْضُ ٱلْخِبْرَةِ فِي ٱلْعَمَلِ عَلَىٰ ٱلْفُلِّ سْتَاكِ عِنْدَ ٱلْحَاجَةِ.

حَالِيًّا أَعْمَلُ عَنْ بُعْدٍ كَمُطَوِّرِ مُوبَايِل فِي شَرِكَةِ سَيَّارَة، وَهِيَ شَرِكَةٌ نَاشِئَةٌ فِي ٱلْأُرْدُنِّ.

حَالِيًّا

فِي الْآوِنَةِ الْأَخِيرَةِ، أَعْمَلُ عَلَى PHP Laravel فِي بَعْضِ الأَعْمَالِ، فِي مَشْرُوعٍ تَعَاقُدِيٍّ يَتَضَمَّنُ video streaming feature. وَبِجَانِبِ ذٰلِكَ، أَتَعَلَّمُ backend development بِشَكْلٍ مُنْتَظِمٍ، مِثْلَ PostgreSQL وَ Docker وَ Go وَ PHP وَ Amazon AWS وَغَيْرِهَا، وَأَدْرُسُ أَيْضًا دُرُوسًا فِي QA testing وَ UI UX عَلَى Udemy.

خَارِجَ الْعَمَلِ، الْقِرَاءَةُ هِيَ مَصْدَرُ تَرْفِيهِي الْوَحِيدُ. أَقْرَأُ فِي الْغَالِبِ رِوَايَاتٍ لِلْيَافِعِينَ وَ رِوَايَاتٍ مُغَامِرَاتٍ وَ رِوَايَاتٍ مُثِيرَةً، وَأَحْيَانًا أَقْرَأُ فِي الفَلْسَفَةِ وَالتَّارِيخِ وَ العَقِيدَةِ الإِسْلَامِيَّةِ وَ السِّيَاسَةِ. كَمَا أُوَاكِبُ التِّقْنِيَّةَ وَالأَحْدَاثَ الْمُعَاصِرَةَ مِنْ خِلَالِ الْمُدَوَّنَاتِ، وَأَقْضِي وَقْتًا غَيْرَ صِحِّيٍّ كَثِيرًا عَلَى Hacker News.
